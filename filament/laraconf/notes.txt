//can quickly make models with laravel blueprint (yml file)
https://blueprint.laravelshift.com/
composer require -W --dev laravel-shift/blueprint
php artisan blueprint:new

//remove created files 
php artisan blueprint:erase

after draft.yml file is setup run with 
php artisan blueprint:build
builds factories, migrations and models 

//publish stubs
php artisan blueprint:stubs
creates template stubs for bluprint that I can modify for creating blueprints

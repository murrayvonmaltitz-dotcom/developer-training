     * Bootstrap any application services.

    public function boot(): void
    {
        Gate::define('view-admin', function (User $user) {
            return true;
        });
    }

//policies are the controller equivilent to gates. they allow us to assign authorisation specifically for a model

corrisponding model: Idea
found under app/policies

example 
    public function update(User $user, Idea $idea): bool
    {
        //check if logged in user id matches that to the user id on the current idea
        return $user->id === $idea->user_id;
    }
used in controller 

   public function show(Idea $idea)
    {
        Gate::authorize('update', $idea);

        return view('ideas.show', [
            'idea' => $idea
        ]);
    }
